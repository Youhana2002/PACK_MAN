Title: Pac-Man Game 🕹️Project DescriptionThis project is an implementation of the classic Pac-Man game, developed using Java. The project focuses on simulating core game mechanics, including player movement, ghost AI, and a point-tracking system.Technical FeaturesDeveloped using Java with a strong emphasis on Object-Oriented Programming (OOP) principles.  Organized assets by utilizing a dedicated resources folder, ensuring a clean separation between game logic and design elements.  Configured for compatibility with major Java IDEs (IntelliJ IDEA, Eclipse, NetBeans) by employing a comprehensive .gitignore file to maintain a clean repository.  How to Run the GameClone or download the repository to your local machine.  Open the project folder using any Java-compatible IDE (e.g., IntelliJ IDEA).  Ensure the Source Path is configured to include the src directory.  Run the main entry point: src/App.java.
